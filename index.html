Qu'est-ce qu'une adresse ip?

  • Adresse IP definition
      • Une adresse ip est le nom donner a l'adresse electronique d'un objet connecté.
    IP veut dire internet protocol ceci est une chaine de caractères.

    l'adresse peut etre privé on l'appelera IP lan ou elle peut etre publible on l'appelera alors IP wan.
    Là ou arrive votre ligne internet, votre routeur ,possède un IP wan,le routeur qui fais parvenir internet a vos objet connecté qui eux possède une IP lan

  • voici quelque exemples illutrer des objet qui peuvent possèder des IP wan et lan ainsi que les IPv4 et IPv6.

    • Il y a aussi plusieurs variantes des adresses IP, les IPv4 et les IPv6.

    1. les IPv6 sont des colbinaison quasiment illimités.
    2. Les IPv4 est constituée de 4 groupes de chiffres allant de 0 a 254.


    Qu'est-ce qu'une adresse MAC?

    Maintenant nous allons expliquer ce que sont les adresses MAC.


      MAC veut dire Media access control
    1. Chaque appareil pouvant se connecter est identifier de manière unique par une adresse MAC
    2. il s'agit d'un identifiant distinct de l'adresse IP,qui quant a elle sers a localiser un appareil dans un réseau
    3. l'adresse MAC est composé de 12 caractères héxadécimaux*
    4. elle se constitu de 2 groupes de 6 caractères héxadécimaux*
    5. ces 2 groupes de 6 caractères sont généralement séparé par des guillemets ou des tirets

      6. *Se dit d'un système de numération de base 16.

      routeurs

      Un routeur est un appareil permettant de créer un réseau Wi-Fi. Il doit être relié à un modem. Il envoie les informations venant d'Internet à nos appareils personnels (ordinateurs, téléphones, tablettes etc.).
      Ces appareils connectés à Internet chez nous constituent notre réseau local (LAN).
